# OS.js Stylelint Rules

These are the default Stylelint rules for OS.js codebase.

## Installation

```
npm install --save-dev @aaronmeese.com/stylelint-config
```

In your stylelint config:

```json
{
  "extends": "@aaronmeese.com/stylelint-config"
}
